# React Native App

## 👁️ Overview
This project is a basic React Native app created using Expo:
1. Dispay text
2. Border styling
3. Text styling (font, size, color)
4. Spacing (margin & padding)

## ✅How to run it:
* open terminal
* cd testingEnvironment
* npx expo start
* press w key to oppen in browser
